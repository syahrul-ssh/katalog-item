## Tech Stack

- **Node.js**
- **NestJS**
- **TypeScript**
- **SQLite (TypeORM)**
- **Jest (Unit Testing)**
- **Docker & Docker Compose**

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Set up the database

```bash
npm run migration:run
```

### 3. Start the application

```bash
npm run start:dev
```

### 4. Start the application with Docker

#### Build & Run Application
```bash
docker compose up --build
```
